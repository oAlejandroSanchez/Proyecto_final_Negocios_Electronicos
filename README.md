# Proyecto_final_Negocios_Electronicos
La idea de este proyecto es darle una segunda oportunidad a los equipos que se desechan de empresas y organismos. Esto con el objetivo de reducir la contaminación producida por los equipos de cómputo y permitiéndoles a las escuelas de sitios marginales integrarse al sistema de educación virtual.

# Tecnologías utilizadas
 * React
